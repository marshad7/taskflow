# Taskflow

![CI](https://github.com/marshad7/taskflow/actions/workflows/ci.yml/badge.svg)

Taskflow is a full-stack task management web application built with Node.js, Express, and PostgreSQL.

## Planned Features
- User authentication (session-based)
- Task CRUD (create, edit, delete)
- Status tracking: Todo / Doing / Done
- Priority levels and due dates
- Filters and search

## Tech Stack
- Node.js
- Express
- PostgreSQL
- bcrypt
- express-session
- Docker (planned)
- GitHub Actions CI (planned)

## Run Locally
npm install
npm run dev

Work in progress (actively being developed)
